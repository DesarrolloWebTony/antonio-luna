Descripción general del puesto

En conjunto con UMB plantel Atenco, se desarrolló un blog para el municipio de Texcoco. Siguiendo un patrón de diseño conservando la calidad del contenido, el antiguo sitio no tenía un diseño responsivo esto se adaptó logrando más vistas.

  • Análisis y Modelado usando mock-data como alternativa a una BD.
  • Revisión del contenido de cada artículo.
  • Unificación y generación de archivos para el deploy a producción usando GitHub Pages.
  • Poniendo en marcha el uso de flexbox para la adaptación a dispositivos móviles.
